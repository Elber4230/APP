# ELBER-

Mini app online para crear collages de fotos desde el navegador.

## Uso rápido

1. Abre `app/index.html` en tu navegador.
2. Sube varias imágenes.
3. Elige diseño automático, tamaño carta y formato de descarga (PDF/PNG).
   - En tamaño carta se aplica un margen izquierdo de 0,5 cm para impresión.
4. Genera el collage y descárgalo.
